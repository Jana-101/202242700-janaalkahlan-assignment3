# Personal Web Portfolio – Assignment 3

## 📌 Project Description

This project is an advanced version of the personal portfolio website developed in Assignments 1 and 2.
It focuses on enhancing interactivity, improving user experience, and integrating dynamic data.

The goal of this assignment is to build a more complete, user-friendly, and feature-rich web application using JavaScript and modern frontend practices.

---

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
* GitHub REST API

---

## ✨ Features

### Core Features

* Responsive design for desktop, tablet, and mobile
* Navigation with hamburger menu
* Theme toggle (light/dark mode using localStorage)
* Project filtering by category
* Live search functionality
* Contact form with validation

### Assignment 3 Enhancements

* GitHub API integration (live repositories display)
* Project sorting (A–Z, Z–A, category)
* Persistent state (theme, filter, search saved in localStorage)
* User name personalization with welcome message
* Session timer showing time spent on the website
* “How to Use” guide for better user experience
* Improved user guidance across sections
* Cleaner form UX with real-time validation

---

## 🧭 How to Use the Website (User Guidance)

1. Use the navigation menu to move between sections (About, Projects, Skills, Contact)
2. In the Projects section:

   * Use search to find projects
   * Use filters to view categories
   * Use sorting to organize results
3. Scroll to the GitHub section to view live repositories
4. Toggle between light and dark mode using the top button
5. Fill out the contact form and submit your message
