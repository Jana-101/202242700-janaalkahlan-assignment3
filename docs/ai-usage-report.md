# AI Usage Report – Assignment 3

## Tools Used & How They Were Used

### ChatGPT

ChatGPT was mainly used during planning, setup, and refinement.

* I started by creating a general idea of my portfolio and features.
* I used ChatGPT to:

  * Set up the GitHub repository and project structure
  * Improve and expand my initial ideas
  * Suggest more relevant and advanced features for Assignment 3
  * Provide guidance on improving user experience and usability
  * Help refine documentation

ChatGPT was also used to evaluate and review the project and suggest improvements in clarity, usability, and structure.

---

### Claude

Claude was mainly used during the coding and implementation phase.

* I provided my initial code, which was not fully organized.
* Claude helped to:

  * Clean and restructure the code
  * Improve readability and consistency
  * Improve CSS styling and layout
  * Add new features (such as GitHub API integration and sorting)
  * Fix bugs and improve functionality
  * Enhance user interface elements

---

## How AI Was Used in the Workflow

1. I first planned the website structure and features myself
2. I used ChatGPT to improve and expand those ideas
3. I implemented initial code
4. I used Claude to refine, clean, and extend the code
5. I reviewed all outputs and adjusted them to fit requirements

AI tools were used as assistants throughout development, not as a replacement for my own work.
